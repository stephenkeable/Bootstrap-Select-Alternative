Bootstrap Select Alternative

After seeing Alice Bartlett's great talk about select tags, we built this alternative UI for bootstrap.

Requires

Bootstrap
jQuery
Font Awesome

Example usage

Demo version

<div class="radio-select-container panel panel-default">
<!-- Header with collapse action on click -->
<div class="panel-heading"><a href="#"><i class="fa fa-fw fa-chevron-up pull-right"></i><span>Collapse Menu</span></a>
</div>
<!-- List items container -->
<div class="radio-select-inner panel-body">
    <!-- Example item -->
    <div class="radio-wrapper radio">
        <label>
            <input type="radio" name="optionsRadios"><span>Label text in here</span>
        </label>
    </div>
    <!-- Example item END -->
</div>
<!-- List items container END -->
</div>
